Quick Start

1
Setup
Run in your project directory. Choose your platform when prompted (Claude Code / OpenClaw / WorkBuddy).
curl -fsSL https://raw.githubusercontent.com/Trust-App-AI-Lab/StudyClawHub/main/setup.sh | bash
2
Create
Scaffold a Skill, create a GitHub repo, and push — all guided by the agent.
/sch-create
3
Submit
Register your Skill on StudyClawHub via a GitHub Issue.
/sch-submit
4
Discover
Find and install Skills created by others.
/sch-search /sch-install
